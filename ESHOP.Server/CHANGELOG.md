Tento soubor vysvětluje, jak Visual Studio vytvořilo projekt.

Při vytváření tohoto projektu byly použity následující kroky:
- Vytvořte nový projekt webového rozhraní API Core ASP\.NET.
- Aktualizujte soubor projektu a přidejte odkaz na front-endový projekt a nastavte vlastnosti SPA.
- Aktualizujte `launchSettings.json`, aby se proxy SPA zaregistrovalo jako spouštěcí sestavení.
- Přidejte projekt do seznamu projektů po spuštění.
- Zapište tento soubor.
